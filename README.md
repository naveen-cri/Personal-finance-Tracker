# Personal Finance Tracker

A simple web application for tracking income and expenses.

## Features

- Add income and expense transactions.
- View totals for income, expenses, and balance.
- See a running list of transactions.

## Run locally

Because this is a static app, you can run it with any static file server.

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000> in your browser.
